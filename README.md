# Carmel by the Sea Ministry

A serene, faith-centered website for grief support and prayer ministry.

## Overview
This is a simple, secure, single-page static website built with HTML, CSS (Tailwind), and vanilla JavaScript. It features a calming coastal design, prayer request form, donation links, and clear ministry messaging.

## Quick Start

### Preview Locally
1. Download or clone the repository.
2. Open `index.html` in any web browser.
3. Edit directly in VS Code (or any text editor) — changes appear instantly on refresh.

### Deploy Live (Free Options)

**GitHub Pages (Recommended for simplicity)**
1. Ensure this repo is public.
2. Go to **Settings → Pages**.
3. Under "Source", select **Deploy from a branch** → `main` → `/ (root)`.
4. Save. Your site will be live at:  
   `https://MichalWyatt.github.io/carmel-by-the-sea-ministry/`

**Netlify (Alternative)**
1. Drag and drop the project folder onto [netlify.com](https://netlify.com).
2. Or connect this GitHub repo for automatic deploys on every push.

## Making Updates
- Edit `index.html` for content, colors, or sections.
- Commit and push to the `main` branch — changes go live automatically.
- For the prayer form: It’s currently set up for Netlify Forms. Switch to Formspree or another service if using GitHub Pages.

## Customization Tips
- Replace placeholder text, images, and contact info with your real ministry details.
- Update colors in the Tailwind classes at the top of `index.html`.
- Add your logo and real photos for a personal touch.
- For donations: Replace buttons with Stripe or PayPal links.

## Tech Stack
- Pure static HTML/CSS/JS (no build step needed)
- Tailwind CSS via CDN
- Mobile-responsive design
